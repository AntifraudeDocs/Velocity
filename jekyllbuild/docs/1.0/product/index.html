Produto

Permite ao VENDEDOR cadastrar e gerenciar seus produtos. Cada produto possui uma coleção de SKUs.

Atributos do objeto

Id int
Identificador do produto.

CreatedOn datetime
Data da criação do objeto

UpdatedOn datetime
Data da última atualização do objeto

MerchantId guid
Identificador da loja, composto por 36 caracteres no formato 99999999-9999-9999-9999-999999999999

Name 256 string
Nome do produto

Description 2048 string
Descrição do produto

IsEnabled bool
Flag que indica se produto está habilitado

Manufacturer.Name opcional 64 string
Fabricante do produto

Manufacturer.Model opcional 64 string
Modelo do produto

Manufacturer.Warranty opcional int
Garantia do produto (em meses)

Skus opcional array
Lista de SKUs do produto

Operações HTTP

POST /api/product/
Criação de produto

PUT /api/product/{productId}
Atualização de produto

GET /api/product/{productId}
Obtenção de Produto por Id

GET /api/product/
Listagem de Produtos

POST Criação de Produto

REQUEST:

POST /api/product/ HTTP/1.1
Host: {blackbox endpoint}
Authorization: Bearer {access_token}
Content-Type: application/json

{
  "MerchantId": "99999999-9999-9999-9999-999999999999",
  "Name": "Nome do produto",
  "Description": "Descrição do produto",
  "Manufacturer": {
    "Name": "Fabricante do produto", 
    "Warranty": 12,
    "Model": "MODEL0001"
  }
}

RESPONSE:

HTTP/1.1 201 Created
Content-Type: application/json;charset=UTF-8

{
  "Id": 9999,
  "Links": {
    "self": {
      "href": "/api/product/9999",
      "method": "GET"
    }
  }
}
